# 01 - LLM Fundamentals

> The foundation of the AI-Platform-Lab learning path.

## Overview

This module introduces the core concepts behind Large Language Models
(LLMs). By the end of this module you will understand what LLMs are, how
they work internally, their strengths and limitations, and how they are
used in modern AI applications.

Unlike many tutorials that focus only on APIs, this module explains the
engineering concepts that power systems like ChatGPT, Claude, Gemini,
Llama, Mistral, Qwen and DeepSeek.

## Learning Objectives

After completing this module you will be able to:

-   Explain what an LLM is.
-   Describe the Transformer architecture at a high level.
-   Understand tokens and tokenization.
-   Explain embeddings and context windows.
-   Differentiate training from inference.
-   Understand hallucinations and why they happen.
-   Compare commercial and open-source models.
-   Prepare for Prompt Engineering and RAG.
