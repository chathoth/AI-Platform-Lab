# Contributing

This is a personal learning repository documenting my hands-on AI/LLM
journey from a platform engineering perspective. It isn't run as a
formal open-source project, but issues and suggestions are welcome.

## Reporting an issue

If you spot something wrong - a broken example, an inaccurate claim,
an outdated command - please open a GitHub issue. Include:

- Which module/chapter or file is affected
- What you expected vs. what actually happened
- Your environment (OS, Python version, Ollama version) if relevant

## Suggesting a change

Pull requests are welcome for fixes and improvements. To keep this
repository consistent with how it's written so far:

- **Verify before documenting.** Every claim about behavior in this
  repo (a command's output, an API's response, an error message) is
  backed by actually running it, not assumed. If you're proposing a
  doc change, run the corresponding code first.
- **Keep the platform-engineering framing.** Explanations favor
  practical DevOps/SRE examples over abstract theory.
- **Match the existing structure.** Each module follows the same
  `docs/` + `examples/` (+ `notebooks/` where present) layout - see
  any completed module for the pattern.

## Questions

Open an issue - happy to discuss.
